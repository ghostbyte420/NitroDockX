## üìå **How to Use NitroDockX**
---

### 1. Install and Run
- Download and run **NitroDockX.exe**.
- The dock will appear on your screen (by default, on the right side).

### 2. Add Icons
- **Drag and drop** any file, folder, or app onto the dock.
- Right-click an icon to **customize or remove** it.

### 3. Customize the Dock
- Right-click the **config icon** (üîß) to open **Settings**.
- Adjust:
  - **Position** (left, right, top, bottom).
  - **Icon size and spacing**.
  - **Opacity and glow effects**.
  - **Skins and textures**.

### 4. Save Your Setup
- All your changes are **automatically saved** to `NitroDockX.ini`.
- The dock can **start with Windows** (enable in Settings). <br> <br>


## üîß Advanced Tips
---

### Reorder Icons
- **Middle-click** an icon and **scroll** to move it up or down.

### Change Skins
- Drop your own images into the `NitroSkins` folder and select them in Settings.

### Use on Multiple Monitors
- Assign the dock to any monitor in **Settings > Assign Monitor**.

### Reset the Dock
- Clear the `.ini` file to start fresh (backup first!). <br> <br>
